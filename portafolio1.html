En esta segunda lección de nuestro tutorial de HTML vamos a ver un poco de información basica de HTML, para comenzar, HTML significa (Hyper Text Markup Language) es decir, lenguaje de marcado de hiper texto, una vez hemos conocido esto vamos a ver las etiquetas que son esenciales para tener un doc de HTML con unba semantica correcta


(!DOCTYPE html), esto le indica a la computadora que estamos especificando un doc. HTML, es decir que siempre debemos poner esto, es muy esencial

(html lang="en"), esto le indica a la computadora que el doc de HTML que vamos a hacerlo en este caso en ingles pues "en"son las siglas de "en"glish, por eso siempre debemos ponerlo, pues asi especificamos que idioma vamos a utilizar

Los meta, los meta son muy importantes a la hora de que nuestro doc. HTML tenga una semantica correcta, alguos de los meta más usados que yo he puesto en este do. Son:
(meta charset="utf-8") (meta name="viewport" content="width=device-width, initial-scale=1") (meta name="Portafolio" content="Este es un portafolio de un proyecto de HTML Y CSS") (meta name="author" content="Nicolás Boada López") (meta name="robots" content="noindex")

El (meta charset="utf-8") le indica al navegador web que el contenido del documento está codificado utilizando UTF-8, que es un estándar de codificación de caracteres que admite una amplia gama de caracteres de varios idiomas y símbolos, sin esto, la letra saldría mal, por ejemplo: PaG1n@, y no queremos que salga así, sin duda queremos que salga: Página. Por eso es muy importante ponerlo.

El (meta name="viewport" content="width=device-width, initial-scale=1") sirve para que nuestra pagina sea responsive, es decir, que se adapte a cualquier tipo de pantalla, ya sea una tablet, un celular, un computador, etc... Es decir que siempre debemos poner esto.
El (width=device-width) sirve para configurar la anchura inicial de la ventana gráfica del navegador en dispositivos móviles. Esta configuración es crucial para el diseño responsivo de sitios web, asegurando que el contenido se adapte adecuadamente al tamaño de la pantalla del dispositivo del usuario, ews decir que nos indica que cuandop se abra la página en cualquier dispositivgo el ancho se va a adaptar a la pantalla, por eso, siempre hay que ponerlo.

El (initial-scale=1) especificar el nivel de zoom inicial cuando se carga una página web en dispositivos móviles.
